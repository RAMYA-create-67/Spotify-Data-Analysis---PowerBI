# Spotify-Data-Analysis---PowerBI

##  Project Overview
This project analyzes Spotify song data to uncover insights about music trends, popular artists, and audio features.

## Tools Used
- Power BI
- Power Query

## Dataset
- Contains 5000+ Spotify songs with features like popularity, energy, danceability

## Key Insights
- Identified top artists based on popularity
- Analyzed trends across genres and audio features
- Found patterns in high-performing songs
